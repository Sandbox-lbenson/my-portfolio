# My Portfolio

This portfolio showcases my work across enterprise identity support, production software, developer tools, and infrastructure.

## Features

- Responsive design
- Easy navigation
- Project case studies for selected work
- Refreshed resume page
- Motion-driven loading splash and page transitions
- Public-safe production project writeups

## Technologies

This project was built with the following technologies:

- Vite
- React
- React Router
- Framer Motion
- Sass

## Live Preview

You can view the live version of this portfolio at https://lewis-benson.io.

## Setup

To run this project locally, follow these steps:

1. Clone this repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
4. Open `http://127.0.0.1:5173/` to view the portfolio in the browser.

## Deployment

To deploy this project, follow these steps:

1. Create a production build with `npm run build`
2. Deploy the generated `dist/` folder to your static hosting provider.
